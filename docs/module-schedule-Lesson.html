Lesson

schedule~ Lesson

Класс, хранящий информацию об времени начала и окончания занятия. Предполагается, что каждый день занятия начинаются и заканчиваются в одно и то же время.

Methods

setEndTime(time)

Source:
  • schedule.js, line 70
Устанавливает время конца занятия и проверяет его корректоность:
  • оно не должно быть меньше времени начала занятия если оно было установлено
  • оно не должно быть меньше времени предыдущего исзвесного времени конца занятия
Parameters:
Name Type Description
time number Время начала занятия в минутах, прошедших с полуночи
